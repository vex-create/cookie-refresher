# 🔁 Refresh Roblox Cookie (PHP)

**Quick and simple method.**

### 🛠 What you need:

* XAMPP or any local server
* PHP installed

### 🚀 How to use:

1. Drop the PHP file in your server folder (e.g. `htdocs`)
2. Open in browser:

   ```
   http://localhost/yourfile.php?cookie=YOUR_COOKIE_HERE
   ```
